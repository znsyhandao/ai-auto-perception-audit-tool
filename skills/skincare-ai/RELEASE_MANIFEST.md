## 安装说明

### 系统要求
- **Python**: >= 3.8
- **OpenClaw**: >= 2026.3.0
- **操作系统**: Windows, Linux, macOS
- **内存**: >= 512MB
- **磁盘空间**: >= 50MB

### 安装步骤
```bash
# 方法1: 从ClawHub安装 (推荐)
openclaw skill install skincare-ai

# 方法2: 手动安装
1. 解压此包到OpenClaw技能目录
2. 运行 install.bat (Windows) 或 install.sh (Linux/macOS)
3. 重启OpenClaw
4. 验证安装: /skincare help
```

## 使用限制

### 安全限制
- ❌ **不支持URL**: 只接受本地文件路径
- ❌ **路径限制**: 文件必须在技能目录或允许目录内
- ❌ **文件大小**: 单个文件不超过50MB
- ❌ **文件类型**: 仅支持图像格式 (.jpg, .png, .gif等)

### 推荐做法
- ✅ 将图像文件放在技能目录内
- ✅ 使用相对路径而不是绝对路径
- ✅ 定期检查技能更新
- ✅ 在生产环境使用前先测试
