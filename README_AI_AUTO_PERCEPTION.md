# AI自动感知进化审核工具

## 🎯 项目简介

**AI自动感知进化审核工具**是一个能够自动感知对话中的问题、自动学习经验教训、自动升级审核框架的智能系统。

## 🤖 核心特性

### **1. AI自动感知**
- 自动扫描记忆文件 (`memory/YYYY-MM-DD.md`)
- 自动提取 `/remember` 经验教训
- 自动识别问题模式和解决方案

### **2. AI自动学习**
- 从对话和记忆文件中学习经验
- 建立和维护知识库 (`ai_knowledge_base_v2.json`)
- 计算置信度和相关性

### **3. AI自动进化**
- 自动升级审核框架包含新经验
- 自动添加新的检查规则
- 自动验证升级结果

### **4. AI自动修复**
- 自动修复版本不一致问题
- 自动清理缓存文件
- 自动验证英文合规性

## 📁 文件结构

```
AI_AUTO_PERCEPTION_TOOL/
├── AI_AUTO_SYSTEM.py                    # AI主程序
├── enhanced_audit_framework_v3_fixed.py # 增强版审核框架
├── ai_knowledge_base_v2.json           # AI知识库
├── pre_release_cleaner.py              # 发布前清理器
├── permanent_audit_ascii.py            # 永久审核框架
├── README_AI_AUTO_PERCEPTION.md        # 本文档
└── start_ai_auto_perception.bat        # 启动脚本
```


## 🔧 核心功能详解

### **AI自动感知**
系统会自动扫描 `~/.openclaw/workspace/memory/` 目录下的记忆文件，提取经验教训。

### **知识库系统**
- **问题模式**: 存储识别到的问题模式
- **经验教训**: 存储学习到的经验教训
- **框架升级**: 记录审核框架的升级历史

### **自动修复功能**
1. **版本一致性检查**: 确保ZIP文件名与技能版本匹配
2. **缓存文件清理**: 自动清理 `__pycache__` 和 `.pyc` 文件
3. **英文合规检查**: 验证核心文件100%英文

## 📊 包含的2026-04-03经验教训

系统已学习以下8个关键经验教训：

1. **版本号必须更新** - 每次重大改进都要更新版本号
2. **文件名版本一致** - ZIP文件名必须与技能版本一致 (skill-v1.1.0.zip)
3. **100%英文要求** - ClawHub审核要求所有文件100%英文
4. **完整文档集** - 技能需要SKILL.md, README.md, CHANGELOG.md
5. **功能测试必须** - 发布前必须测试所有命令
6. **清理缓存文件** - 发布前必须清理__pycache__和.pyc文件
7. **安全验证** - 确保无危险函数，无网络访问
8. **细节检查** - 检查所有文件名、版本引用和文档一致性

## 🔍 技术架构

### **核心模块**
1. **感知模块**: 扫描记忆文件，提取经验教训
2. **学习模块**: 建立和维护知识库
3. **审核模块**: 运行增强版审核框架
4. **修复模块**: 自动修复发现的问题
5. **进化模块**: 升级审核框架

### **数据流**
```
对话/记忆文件 → AI感知 → 学习经验 → 升级框架 → 审核技能 → 修复问题 → 生成报告
```
